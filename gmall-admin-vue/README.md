# gmall-admin-vue
谷粒商城后台管理系统前端vue项目
### 前端项目启动步骤：
* 首先安装 node.js http://nodejs.cn/download/
    无脑 next 注意安装目录
* 安装 cnpm  命令： npm install -g cnpm --registry=https://registry.npm.taobao.org
* 检查是否安装成功 node-v  npm-v cnpm-v

* 然后 在项目目录下打开 cmd
* 运行命令 npm install （基本都不行）
如果报错 使用  cnpm install

-启动项目 ： npm/cnpm start
- 关闭项目： 请在 cmd 窗口直接 Ctrl + C
